# MathAndPhsical Skill

这是一个面向初高中数学、物理题目可视化讲解视频制作的 Codex skill，主要用于把一道完整题目转化为清晰、分步骤、可审阅、可合成的 Manim 解析视频。

## 适用场景

- 初中数学、高中数学题目讲解
- 初中物理、高中物理题目讲解
- 函数图像、几何图形、物理模型、运动过程等可视化
- 纯公式推导、证明题、计算题的视频化讲解
- 从一道题目出发，生成完整解析视频

## 工作流

1. 先解析题目，识别已知条件、小问和教学目标。
2. 先写完整解答文案，要求清晰、严谨、适合课堂讲解。
3. 等用户确认文案后，再开始制作视频。
4. 按小问拆分 Manim 视频片段。
5. 先渲染 480p 预览，供用户审阅内容、节奏、布局和公式可读性。
6. 用户确认后，再渲染 1080p 正式版。
7. 生成统一风格的 4:3 视频封面图。
8. 在正式讲解前加入原题展示段。
9. 合并题目展示段和所有讲解片段，形成完整解析视频。

## 画面布局

### 可视化讲解型

- 上方显示当前小题的原始题目文本。
- 左侧作为主要可视化区域，用于图像、几何图形、坐标系、力学模型等。
- 右侧展示推导、证明步骤和关键公式。
- 避免图形标签、公式、文字互相重叠。
- 推导较长时分屏展示，不把文字压到边缘。

### 纯推导讲解型

- 上方显示当前小题的原始题目文本。
- 中央使用推导板展示主要计算或证明过程。
- 逐行揭示公式和逻辑，必要时分阶段清屏。
- 不强行加入无意义的图形区域。

## 封面风格

默认生成 4:3 封面图，推荐尺寸为 1440x1080。

- 深色背景
- 大号居中文字
- 第一行白色，例如 `2026高考数学`
- 第二行暖黄色，例如 `新一卷第18题`
- 背后使用淡蓝色椭圆或坐标轴作为数学氛围元素
- 不放完整题目、解题步骤或额外说明
